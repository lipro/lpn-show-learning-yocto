Package Area
============

.. figure:: /build-system/concepts/package-feeds-area.*
   :name: package-feeds-area
   :figclass: align-center
   :align: center

   :ref:`ypdoc:overview-manual/concepts:package feeds`
   as part of Yocto Build System Workflow

When the OpenEmbedded build system generates an image or an SDK, it gets
the packages from a package feed area located in the Build Directory. The
general workflow figure shows this package feeds area in the upper-right
corner. See :doc:`/package-feeds/index` for more details.
